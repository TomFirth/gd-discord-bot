# GD Bot

A Discord bot for game developers.

## Future ideas

- `challenge` — weekly or random development challenge prompt
- `devtip` — a random game development tip or best practice
- `assetidea` — suggest an art or sound asset concept
- `moodboard` — suggest a genre + tone + color scheme combo
- `showcase` — feature a random indie game developer showcase (potentially from Reddit)
- `boss` — generate a boss concept with attack style and mechanics
- `audio` — suggest a sound effect idea for a situation
- `story` — generate a one-line lore hook or story prompt
